  <name>Task 1: Create amc-utils.js shared utility module</name>
  <files>site/js/amc-utils.js</files>
  <action>
Create `site/js/amc-utils.js` as a single IIFE exposing `window.AMC_UTILS`. Follow the exact IIFE pattern from `components.js` with decorative header comment. Use `var` (not `const`/`let`) -- this is an ES5 codebase.

The file must contain these four sections in order:

**1. Formatters section (`// -- Formatters --`):**
- `AMC_UTILS.fmt(n)` -- Dollar millions: `$1,234.5M`, parentheses for negatives, `'N/A'` for null/undefined/NaN. Use `toFixed(1)` and regex comma insertion.
- `AMC_UTILS.fmtPct(n)` -- Raw percentage input (e.g., 85.3 -> `'85.3%'`). `toFixed(1)`, `'N/A'` for null/undefined/NaN.
- `AMC_UTILS.fmtPctDecimal(n)` -- Decimal input (e.g., 0.853 -> `'85.3%'`). Multiplies by 100 then delegates to `fmtPct`. For the `financials-model.js` convention.
- `AMC_UTILS.fmtX(n)` -- Leverage multiple: `'5.50x'`. `toFixed(2)`, `'N/A'` for null.
- `AMC_UTILS.fmtDollar(n)` -- Plain dollar: `'$1,234.50'` (no M suffix). `toFixed(2)`, parentheses for negatives.
- `AMC_UTILS.fmtShares(n)` -- Share count: `'1,234.56'`. `toFixed(2)` with comma insertion.
- `AMC_UTILS.fmtInt(n)` -- Integer with commas: `'1,234'`. For counts, no decimal. `'N/A'` for null.
- `AMC_UTILS.colorVal(n)` -- Returns `'#ef4444'` for negative, `'#22c55e'` for non-negative.

All formatters must guard: `if (n === null || n === undefined || isNaN(n)) return 'N/A';`

**2. Chart Defaults section (`// -- Chart Defaults --`):**
- Private `var _defaultsInitialized = false;` guard.
- `AMC_UTILS.initChartDefaults = function()` that:
  - Returns immediately if `_defaultsInitialized` is true.
  - Sets `_defaultsInitialized = true`.
  - Reads CSS custom properties via `getComputedStyle(document.documentElement)` for `--text-muted` and `--border`, with fallbacks `'#8899b4'` and `'#2a3654'`.
  - Sets `Chart.defaults.color`, `Chart.defaults.borderColor`, `Chart.defaults.font.family` (SF Mono, Fira Code, JetBrains Mono, monospace), `Chart.defaults.font.size` (11), `Chart.defaults.responsive` (true), `Chart.defaults.maintainAspectRatio` (false).
  - Sets tooltip defaults: backgroundColor `'rgba(17, 24, 39, 0.95)'`, titleFont/bodyFont with monospace family and size 11/10, borderColor from CSS, borderWidth 1, padding 8.
  - Sets legend defaults: labels.font size 10 monospace, labels.padding 12, labels.usePointStyle true.
- Auto-execute `AMC_UTILS.initChartDefaults()` at the end of the IIFE body (after all definitions, before `window.AMC_UTILS = AMC_UTILS;`). This matches existing behavior where defaults are set at script load time.

**3. ChartRegistry section (`// -- ChartRegistry --`):**
- `AMC_UTILS.ChartRegistry` object with:
  - `_instances: {}` (private store).
  - `set(key, chartInstance)` -- If `_instances[key]` exists, call `.destroy()` on it. Store `chartInstance` at `_instances[key]`. Return `chartInstance`.
  - `get(key)` -- Return `_instances[key] || null`.
  - `destroy(key)` -- If exists, call `.destroy()` and `delete`. No-op if not found.
  - `destroyAll()` -- Iterate all keys, destroy each, reset `_instances = {}`.
  - `has(key)` -- Return `!!_instances[key]`.
  - `keys()` -- Return `Object.keys(_instances)`.

**4. DataLoader section (`// -- DataLoader --`):**
- `AMC_UTILS.DataLoader` object with:
  - `_cache: {}` (stores resolved data AND pending Promises for deduplication).
  - `_basePaths: ['../data/', './data/', 'data/']` (the three path trials from financials-model.js).
  - `fetch(filename)` -- If `_cache[filename]` is a Promise or resolved data, return `Promise.resolve(_cache[filename])` (handles both in-flight dedup and resolved cache). Otherwise, create the Promise chain with path-trial fallback, store the Promise immediately in `_cache[filename]` (before resolution -- this deduplicates concurrent requests), resolve to parsed JSON data, replace cache entry with resolved data. On all-paths-exhausted, reject with descriptive Error and delete cache entry.
  - `clearCache()` -- Reset `_cache = {}`.

Finally: `window.AMC_UTILS = AMC_UTILS;` at end of IIFE.

The file should be approximately 150-200 lines. Use the decorative comment header style: `/* ═══ AMC Debt Navigator - Shared Utilities ═══ */`.
