round_cell() logic (from research):
- Load with `openpyxl.load_workbook(path, data_only=True)` — resolves all formula cells
- `cell.number_format` gives format string e.g. `#,##0`, `#,##0.0`, `0%`, `0.00%`
- For `#,##0` → dp=0, for `#,##0.0` → dp=1, for `0%` → dp=0+2=2 (stored as decimal), for `0.00%` → dp=2+2=4
- `math.isnan()` / `math.isinf()` guard before rounding
- `General` format → return value unrounded (metadata cells)
- datetime values → `.strftime('%Y-%m-%d')`

2. **Path constants**:
   - `EXCEL_PATH = os.path.join(os.path.dirname(__file__), '..', 'AMC BRx.xlsx')`
   - `OUT_DIR = os.path.join(os.path.dirname(__file__), '..', 'site', 'data')`

3. **`round_cell(cell)` function**: Implements number_format-aware rounding per research:
   - Guard: if `val is None` or not `isinstance(val, (int, float))` → check `isinstance(val, datetime)` and return `.strftime('%Y-%m-%d')`, else return `val`
   - Guard: `math.isnan(val) or math.isinf(val)` → return `None`
   - `fmt = cell.number_format or 'General'`
   - If `fmt == 'General'` → return `val` unchanged
   - If `'%' in fmt`: `m = re.search(r'0\.(0+)%', fmt)`, `dp = len(m.group(1)) if m else 0`, return `round(val, dp + 2)`
   - Else: `m = re.search(r'0\.(0+)', fmt)`, `dp = len(m.group(1)) if m else 0`, return `round(val, dp)`

4. **`get_period_columns(ws, header_row, min_year=2019)` function**: Reads the period header row, returns list of `(col_index, period_label)` tuples for columns where `int(cell.value) >= min_year`. Period label is `str(int(cell.value))` for year integers. Skips `None` values.

5. **`extract_flat_array(ws, row_map, period_cols, projected_start=None)` function**:
   - `row_map`: dict of `{output_key: excel_row_number}`
   - `period_cols`: list of `(col_index, label)` from `get_period_columns()`
   - Returns dict with:
     - `"periods"`: list of period labels
     - `"isProjected"`: list of booleans — `True` if label >= `str(projected_start)` (when projected_start is set)
     - `"projectedStart"`: projected_start integer (omit if None)
     - One array per key in row_map: `[round_cell(ws.cell(row=row_num, column=col)) for col, _ in period_cols]`

6. **`extract_sensitivity_grid(ws, row_start, row_end, col_start, col_end, label_col, label_row)` function**:
   - Reads a 2D grid from the sheet
   - `row_axis`: list of cell values from `label_col`, rows `row_start` to `row_end`
   - `col_axis`: list of cell values from `label_row`, cols `col_start` to `col_end`
   - `grid`: 2D array `[[round_cell(ws.cell(r,c)) for c in range(col_start, col_end+1)] for r in range(row_start, row_end+1)]`
   - Returns `{"rowAxis": row_axis, "colAxis": col_axis, "grid": grid}`

7. **`write_json(filename, data)` function**: Writes `json.dumps(data, indent=2)` to `os.path.join(OUT_DIR, filename)`. Creates OUT_DIR if it doesn't exist.

8. **Stub `main()` function**: Loads workbook with `wb = load_workbook(EXCEL_PATH, data_only=True)` and calls (placeholder) extractors. Prints progress. Ends with `print("Extraction complete.")`.

9. **`if __name__ == '__main__':` block** with argparse supporting `--validate` flag (for now, same as default run — validation assertions added in Plan 03).
