<task type="auto">
  <name>Task 2: Implement DCF model JS module with interactive sliders, EV recalculation, and waterfall bridge</name>
  <files>site/js/dcf-model.js</files>
  <action>
Create `site/js/dcf-model.js` as an IIFE module following ES5 conventions (`var`, no arrow functions, `'use strict'`).

**Data Loading:**
- Use `AMC_UTILS.DataLoader.fetch()` for all JSON files
- Load all 5 JSON files in parallel via `Promise.all`:
  - `ufcf.json` (consolidated UFCF + sensitivity grid)
  - `wacc.json` (consolidated WACC)
  - `odeon-dcf.json` (Odeon UFCF)
  - `wacc-odeon.json` (Odeon WACC)
  - `muvico-dcf.json` (Muvico UFCF)
  - `muvico-wacc.json` (Muvico WACC)
- Also load `valuation.json` for equity bridge data (debt, cash values)
- Store in a `DATA` object keyed by entity: `{ consolidated: {ufcf, wacc}, odeon: {ufcf, wacc}, muvico: {ufcf, wacc} }`

**Entity Tab Switching:**
- Track `currentEntity` variable (default 'consolidated')
- On tab click: update `currentEntity`, re-render all panels, update slider defaults to entity's WACC/TGR values
- Use CSS class `.active` on selected tab button

**UFCF Build Table (renderUfcfTable function):**
- Build an HTML table from the entity's UFCF data showing: Period headers, EBIT, Taxes, NOPAT, D&A, CapEx, NWC Changes, UFCF, Discount Period, PV of UFCF
- Use `AMC_UTILS.fmt()` for dollar values, `AMC_UTILS.fmtPct()` for rates
- Highlight projected periods with a subtle accent color

**WACC Decomposition Panel (renderWaccPanel function):**
- Display the WACC calculation components from wacc JSON:
  - Cost of Debt: pretax rate, tax rate, after-tax cost of debt
  - Cost of Equity: risk-free rate, beta, market return, CAPM result
  - Capital weights: debt/equity proportions
  - Final WACC = Wd * Kd * (1-t) + We * Ke
- Use a structured mini-table or definition-list layout

**Terminal Value Panel (renderTvPanel function):**
- Two modes toggled by terminal method selector:
  - Exit Multiple: Show terminal year EBITDA, exit multiple, TV, PV of TV
  - Perpetuity Growth: Show terminal UFCF, TGR, discount rate, TV, PV of TV
- Display sum: PV(UFCF) + PV(TV) = Enterprise Value

**Live EV Recalculation (recalcEV function):**
- Read current WACC from slider (user-adjusted value, NOT from JSON)
- Read current TGR from slider (user-adjusted value)
- Read terminal method toggle state

- **For perpetuity growth method:**
  - Terminal UFCF = entity's last projected UFCF (from JSON array)
  - TV = Terminal_UFCF / (WACC - TGR)
  - Re-discount all projected UFCFs at the new WACC: PV_i = UFCF_i / (1 + WACC)^discountPeriod_i
  - Sum PV of UFCFs
  - PV of TV = TV / (1 + WACC)^lastDiscountPeriod
  - EV = Sum(PV_UFCF) + PV(TV)

- **For exit multiple method:**
  - Use the base exit multiple from JSON (terminalValue.exitMultiple)
  - Terminal EBITDA from JSON
  - TV = Terminal_EBITDA * exitMultiple
  - Re-discount all projected UFCFs at the new WACC (same as above)
  - PV of TV = TV / (1 + WACC)^lastDiscountPeriod
  - EV = Sum(PV_UFCF) + PV(TV)

- Update the EV result box with new Enterprise Value
- Equity bridge: EV - Total Debt + Cash = Equity Value. For consolidated, use valuation.json's debt (4003.5) and cash (477.3). For entities, use entity terminalValue.debt and terminalValue.cash.
- Calculate implied share price: Equity Value / DSO (529.5M shares from wacc.json)
- Color-code equity value (green if positive, red if negative) using `AMC_UTILS.colorVal()`

**UFCF Bar Chart (renderUfcfChart function):**
- Horizontal bar chart (or vertical bar) of UFCF values by period
- Use `AMC_UTILS.ChartRegistry.set('dcf-ufcf', new Chart(...))` to prevent leaks on tab switch
- Bloomberg dark theme (inherits from initChartDefaults)
- Tooltip showing period and UFCF value formatted with `AMC_UTILS.fmt()`

**Sensitivity Grid (renderSensitivityGrid function):**
- Only render for Consolidated entity (Odeon/Muvico have no grids in their JSON)
- Build HTML table from `exitMultipleWaccGrid`: rows = exit multiples, cols = WACC values
- Format column headers as percentages (the colAxis values are half-WACC in the JSON -- note: colAxis is [0.1072, 0.1022, ...] which are WACC/2 values used in the grid. Display them as-is as percentages since that is how the Excel shows them)
- Highlight the cell closest to the current WACC slider value and base exit multiple
- Values formatted with `AMC_UTILS.fmt()` (they are EV in $M)
- If entity has no grid, show "Sensitivity grid available for Consolidated entity only" message

**Waterfall Bridge (setupBridge function):**
- On bridge button click: read current implied EV from recalcEV
- Navigate to `../models/waterfall.html?ev={impliedEV}` where impliedEV is the current enterprise value rounded to nearest integer

**Slider Event Listeners:**
- WACC slider `input` event -> update displayed value, call recalcEV(), update sensitivity grid highlight
- TGR slider `input` event -> update displayed value, call recalcEV()
- Terminal method toggle `click` -> update state, call recalcEV(), update TV panel

**Initialization:**
- On DOMContentLoaded or IIFE bottom: load all data, then call initial render for consolidated entity
- Set slider defaults from JSON values for the consolidated entity

**Important: use `var` throughout (ES5 convention). Use `AMC_UTILS` namespace for all utilities. Do NOT set Chart.defaults directly -- they are handled by amc-utils.js.**
